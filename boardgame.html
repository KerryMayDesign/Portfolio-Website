Board Game

Integrated Design (year 1)

Delivered:

Brand Identity / Game Design / Packaging Design / Rulebook Design

Skills:

Research / Typography / Wireframes / Photography / Miro / Adobe Creative Cloud: Illustrator, Photoshop, InDesign, Bridge / Laser Cutter / Cardboard Prototypes

Project Summary

Design a complete board game including: game board, game pieces, rule book, and packaging. I was encouraged to use an already established game play and redesign it around a pressing social issue of my choice.

My Approach

After researching multiple topics I chose to use totalitarianism as my social issue and used the gameplay of Solitaire. The rulebook helps to set the mood through the use of dictator like speech. The simple packaging materials combined with the graphic symbols and colour choices help to create a feeling of totalitarian rule.

Audience

Adults that like to play one player Solitaire or Chess like games.
